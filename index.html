About the project

3D Gameplay Programming End Of Year Project

This Project was an interesting one. The brief we were given was that we had to make a cube game using OpenGL and SFML. This game had to be a scrolling game where only cubes of the same colour could interact. This was by far one of the more difficult projects I undertook, as I had done no OpenGL before Year 2 of college. I am very proud of the game that I made. I made it so you could not only play the game with keyboard but also that it had controller support. I added a sound track and sound effects for collection of cubes. I ended up getting a First Class Grade in that Module
